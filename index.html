1. получить главную страницу методом GET в протоколе HTTP 1.0;

    получен код главной страницы со статусом 200

  2. получить внутреннюю страницу методом GET в протоколе HTTP 1.1;

    Получен код страницы files/index.php со статусом 404, не найдено

  3. определить размер файла file.tar.gz, не скачивая его;

    Выполнен запрос HEAD и получен ответ Content-Length: 11335

  4. определить медиатип ресурса /image.png;

    Выполнен запрос HEAD и получен ответ Content-Type: image/png

  5. отправить комментарий на сервер по адресу /index.php;

    Выполнен POST запрос с типом application/x-www-form-urlencoded с комментарием hi, получен соответсвующий ответ с передаваемым сообщением

  6. получить первые 100 байт файла /file.tar.gz;

    Выполнен GET запрос, с заголовком Range=0-99 - первые 100 байт. В ответе запрашиваемые байты

  7. определить кодировку ресурса /index.php.

    Выполнен HEAD запрос. Получен ответ charset=UTF-8. Соответственно кодировка UTF-8
